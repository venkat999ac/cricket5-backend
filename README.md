# Cricket Scoreboard App (Python + Kivy)

A simple mobile-friendly cricket scoreboard application built using Python and Kivy.

## Features
- Score buttons: 0, 1, 2, 3, 4, 6
- Extras: LB (Leg Bye), Wd (Wide), NB (No Ball)
- Wicket button
- Popup input for extras
- Automatic score tracking

## Requirements
- Python 3.9 or 3.10
- Kivy 2.3.0

## Install dependencies
```bash
pip install kivy --no-build-isolation
